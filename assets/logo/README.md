# Helios Logo Assets

本目录包含 Helios 编程语言的 Logo 资源。

## 文件说明

- `helios-logo-16x16.png` ~ `helios-logo-1024x1024.png` — 各种尺寸的方形 Logo
- `helios-logo.ico` — 多尺寸 ICO 文件，可用于 Windows 图标
- `helios-banner.png` — 横向 Banner（1200x400）
- `helios-social.png` — 社交媒体预览图（1280x640）
- `design_logo.py` — Logo 生成脚本

## 重新生成

```bash
cd assets/logo
python design_logo.py
```

## 设计理念

Helios 在希腊神话中是太阳神。新版 Logo 用抽象的方式表达"围绕太阳的轨道"，象征能量、秩序与清晰的创造力，同时呈现现代编程语言的专业感。

- **抽象光环**：一道不闭合的渐变轨道替代了卡通的尖锐射线，从青色过渡到靛紫，暗示太阳运行轨迹。
- **太阳亮点**：轨道缺口处放置一颗金色光点，代表 Helios 本身。
- **几何 H 标**：中心为简洁的白色几何字母 "H"，在各种尺寸下保持清晰。
- **深邃圆盘**：深蓝紫渐变的中心圆盘让整体显得稳重、科技感强。
- **通透背景**：方形 Logo 保留透明背景，可适配浅色与深色主题。
- **深色 Banner**：横向 Banner 与社交媒体图采用近黑色深空背景，突出 Logo 的冷色光效。
